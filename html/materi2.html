Penjumlahan Bilangan Bulat

Untuk Menghitung Hasil Operasi Penjumlahan Dua Bilangan Bulat, Kamu Dapat menggunakan garis bilangan. pada garis bilangan, bilangan positif menunjukan kearah kanan, sedangkan bilangan negatif menunjukan kearah kiri agar kamu memahami lebih jelas, perhatikan penjelasan dibawah ini



Perlu di ingat
misalkan a dan b bilangan bulat maka:
a + b = b + a
a + (-b) = a - b
-a + b = -(a - b)
-a + (-b) = -(a + b)

Contoh :
  1. -12 + 5 = -(12 – 5) = -7 → -a + b = -(a – b)
  2. -8 + (-12) = -(8 + 12) = -20 → -a + (-b) = -(a + b)

Sifat-sifat Penjumlahan Bilangan Bulat

Sifat-sifat penjumlahan bilangan bulat sebagai berikut:

  1. Sifat Komutatif

    2. a + b = b + a

    pada penjumlahan dua bilangan bulat berlaku sifat komutatif(pertukaran)
    Contoh :
    -14 + 8 = 18 + (-14) hasilnya -6
    -35 + (-15) = -15 + (-35) hasilnya -50

  2. Sifat Asosiatif

    3. (a + b) + c = a + (b + c)

    pada penjumlahan tiga bilangan bulat berlaku sifat Asosiatif(pengelompokan)
    Contoh :
    (13 + (-19)) + 21 = 13 + ((-19) + 21) hasilnya 15
    (-32 + 14) + (-27) = -32 + (14 + (-27)) hasilnya 9

  3. Sifat Tertutup

    4. jika a dan b bilangan bulat maka a + b = c bilangan bulat

    pada penjumlahan dua atau lebih bilangan bulat menghasilkan bilangan bulat(sifat tertutup)
    Contoh :
    9 + (-13) = -4 → (9,-13 dan -4 adalah bilangan bulat)
    -11 + 38 = 27 → (-11,38 dan 27 adalah bilangan bulat)
