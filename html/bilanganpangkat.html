Pengertian Bilangan berpangkat

Bilangan berpangkat positif adalah suatu bilangan dipangkatkan dengan bilangan bulat. Bilangan berpangkat bulat positif biasa disebut bilangan berpangkat sebenarnya. Bilangan berpangkat diperoleh dari perkalian berulang bilangan itu sendiri. Perhatikan bilangan berpangkat berikut.

  1. \[5^{3} = 5 \times 5 \times 5 = 125\]

  2. \[(-2)^{3} = (-2) \times (-2) \times (-2) = -8 \]

  3. \[3^{5} = 3 \times 3 \times 3 \times 3 \times 3 = 243\]

Umumnya sebuah bilangan real dilambangkan dengan huruf \[a\], sementara bilangan bulat dilambangkan dengan huruf \[n\], maka bilangan berpangkat dapat dituliskan menjadi \[a^{n}\] (dibaca \[a\] pangkat \[n\]), yaitu perkalian \[a\] secara berulang sebanyak \[n\] faktor. Bilangan berpangkat dapat dinyatakan sebagai berikut.


dengan:
\[a^{n}\] : disebut bilangan berpangkat
\[a\] : disebut bilangan pokok
\[n\] : disebut pangkat atau eksponen


Operasi hitung bilangan berpangkat

Bilangan berpangkat dapat dioperasikan dengan bilangan berpangkat lainnya. Untuk lebih jelasnya perhatikan contoh berikut:

  1. \[2^{2} + 4^2 - 3^2 = (2 \times 2) + (4 \times 4) - (3 \times 3)\]

    \[= 4 + 16 - 9 =11\]

  2. \[(-2)^2 \times 3^3 \times 2^4\]

    \[=((-2)\times(-2)\times(-2))\times(3\times3\times3)\]\[\times(4\times4)\]

    \[=(-8) \times 27 \times 16 = -3456 \]

  3. \[6^3 : 3^3 = \frac{6^3}{3^3} = \frac{6 \times 6 \times 6 }{3 \times 3 \times 3} = \frac{216}{27} = 8 \]

Hasil dari operasi bilangan berpangkat tidaklah harus berbentuk bilangan berpangkat juga, hasilnya dapat berupa bilangan bulat atau pecahan.



Sifat bilangan berpangkat

Bilangan berpangkat mempunyai sifat-sifat berpangkat dibawah ini:
  1. \[a^m \times a^n = a^{m+n}\]
    2. Contoh:
    \[2^4 \times 2^3 =2^{4+3}=2^7\]

    \[3^{10} \times 3^5 =3^{10+5}=2^15\]

  2. \[a^m : a^n = a^{m-n}\]
    3. Contoh:
    \[2^6 : 2^4 = 2^{6-4}= 2^2\]

    \[5^7 : 5^2 = 5^{7-2}= 5^5\]

  3. \[(a^n)^m = a^{n√óm}\]
    4. Contoh:
    \[(2^2)^3 = 2^{2\times3}=2^6\]

    \[(4^5)^2 = 4^{5\times2}=4^10\]

tentukan hasil dari \[3\frac{2}{5} \times \] 15%
penyelesaian:
  1. kita ubah semua pecahan diatas menjadi pecahan biasa

    2. \[\frac{3 \times 5 + 2}{5} \times \frac{15}{100}\]

    = \[\frac{17}{5} \times \frac{15}{100}\]

  2. kita kalikan

    3. \[\frac{17}{5} \times \frac{15}{100} = \frac{255}{500}\]

  3. selanjutnya kita sederhanakan

    4. \[\frac{255:5}{500:5} = \frac{51}{100}\]

2 dari 2
tentukan hasil dari \[6\frac{2}{5} : 0,6\]
penyelesaian:
  1. kita ubah semua pecahan diatas menjadi pecahan biasa

    2. \[\frac{6 \times 5 + 2}{5} : \frac{6}{10}\]

    = \[\frac{32}{5} : \frac{6}{10}\]

  2. kita balik pecahan kedua dan ubah operasi bagi menjadi kal

    3. \[\frac{32}{5} : \frac{6}{10} = \frac{32}{5} \times \frac{10}{6}\]

  3. kita kalikan dan sederhanakan

    4. \[\frac{32}{5} \times \frac{10}{6} = \frac{320}{30} = \frac{320:10}{30:10}\]

    \[=\frac{32}{3}\]

    atau
    bisa kita sederhanakan dulu sebelum dikali

    \[\frac{32:2}{5:5} \times \frac{10:5}{6:2} = \frac{16}{1} \times \frac{2}{3}\]

    \[=\frac{32}{3}\]

  4. kita sederhanakan lagi menjadi pecahan campuran
    5. \[=\frac{32}{3} = \frac{30}{3} + \frac{2}{3} = 10\frac{2}{3} \]

    jadi hasil dari \[6\frac{2}{5} : 0,6 = 10\frac{2}{3} \]
