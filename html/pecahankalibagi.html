Perkalian Bilangan pecahan

Misalkan ibu mempunyai kue amparan tatak \[\frac{1}{2}\] bagian. \[\frac{1}{3}\] bagiannya diberikan kepada kakak. jika bagian kue amparan tatak yang diberikan kepada kakak dibandingkan dengan sebuah kue amparan tatak yang utuh, berapa bagiankah yang diberikan kepada kakak? Perhatikan gambar di bawah ini.



Dari gambar di atas terlihat bahwa taman yang ditanami kembang dibandingkan dengan taman keseluruhan adalah \[\frac{1}{6}\] bagian. Sehingga, \[\frac{1}{3}\] dari \[\frac{1}{2}\] adalah \[\frac{1}{6}\], atau \[\frac{1}{3} \times \frac{1}{2} \]=\[\frac{1 \times 1}{3 \times 2}\]=\[\frac{1}{6}\]


Kesimpulan
Misalkan diketahui Pecahan \[\frac{a}{b}\] dan \[\frac{a}{b}\] dengan b ≠ 0 dan d ≠ 0, maka \[\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}\]

Dari kesimpulan tersebut diperoleh :

Untuk b = 1 diperoleh \[\frac{a}{1}\times \frac{c}{d} = \frac{a\times c}{b\times d} \Leftrightarrow a\times \frac{c}{d} = \frac{a\times c}{d}\]

Untuk Perkalian desimal dapat dilakukan dengan cara berikut:
Contoh:
  1. 2,83 x 1,6 =…
    2. \[ 2,83 \times 1,6 = \frac{283}{100} \times \frac{16}{10} = \frac{283 \times 16}{100 \times 10} = \frac{4528}{1000} = 4,528\]

  2. 32,3 x 0,4 =…
    3. \[32,3 \times 0,4 = \frac{323}{10} \times \frac{4}{10} = \frac{323 \times 4}{10 \times 10} = \frac{1292}{100} = 12,92\]

Pembagian Bilangan Pecahan

Pada operasi hitung Bilangan bulat, diketahui bahwa Pembagian merupakan operasi kebalikan dari Perkalian. Bagaimana operasi Pembagian pada Bilangan Pecahan?. Perhatikan Perkalian Bilangan bulat dengan Pecahan berikut.

  1. \[3 \times \frac{1}{2} = \frac{3 \times 1}{2} = \frac{3}{2} = 3 ∶ 2\] sehingga \[3 ∶ 2 = \frac{3}{2} = 3 \times \frac{1}{2}\]

  2. \[5 \times \frac{1}{3} = \frac{5 \times 1}{3} = \frac{5}{3} = 5 ∶ 3\] sehingga \[5 ∶ 3 = \frac{5}{3} = 5 \times \frac{1}{3}\]
    3. jadi dapat disimpulkan bahwa : \[a : b = a \times \frac{1}{b}\]

    Misalkan a = \[\frac{m}{n} dan b = \frac{p}{q}\] maka diperoleh:

    \[\frac{m}{n} ∶ \frac{p}{q} = \frac{m \times q}{n \times p}\]
Pembagian decimal dapat dilakukan dengan cara berikut:
Contoh:
  1. 2,64 : 1,2 =…
    2. \[ 2,64 : 1,2 = \frac{264}{100} : \frac{12}{10} = \frac{264}{100} \times \frac{10}{12} = \frac{2640}{1200} = 2,2\]

    jadi 2,64 : 1,2 = 2,2

  2. 1,33 : 0,3 =…
    3. \[ 1,33 : 0,3 = \frac{133}{100} : \frac{3}{10} = \frac{133}{100} \times \frac{10}{3} = \frac{1330}{300} = 4,43\]

    jadi 1,33 : 0,3 = 4,43

    3. \[\frac{17 \times 20}{5 \times 20} - \frac{15}{100}\]

    \[\frac{17 \times 4}{5 \times 4} - \frac{15 : 5}{100 : 5}\]

    = \[\frac{68}{20} - \frac{3}{20}\]

  3. kita kurangkan

    4. \[\frac{340}{100} - \frac{15}{100} = \frac{325}{100}\]

    atau

    \[\frac{68}{20} - \frac{3}{20}\ = \frac{65}{20}\]

  4. selanjutnya kita sederhanakan

    5. \[\frac{325:25}{100:25} = \frac{13}{4}\]

    atau

    \[\frac{65:5}{20:5} = \frac{13}{4}\]

    pecahan diatas dapat kita ubah menjadi pecahan campuran,desimal maupun persen

    pecahan campuran = \[\frac{13}{4} = \frac{12}{4} + \frac{1}{4} = 3\frac{1}{4}\]

    desimal = \[\frac{13}{4} = 3,25\]

    persen = \[\frac{13}{4} \times 100\] = \[\frac{1300}{4} = 325\]%

ibu mempunyai satu buah kue bingka kemudian di berikan \[\frac{1}{5}\] kepada andi dan 10% dari sisa kue diberikan kepada siti. berapa sisa kue yang dimiliki ibu?
penyelesaian:
  1. kita ubah semua menjadi pecahan biasa
    2. 1 \[-\frac{1}{5} - 10\]% = \[\frac{5}{5} - \frac{1}{5} - \frac{10}{100}\]

  2. kita kurangkan terlebih dahulu \[\frac{5}{5} - \frac{1}{5}\]
    3. \[\frac{5}{5} - \frac{1}{5} - \frac{10}{100} =\frac{4}{5} - \frac{10}{100}\]

  3. kita samakan penyebutnya lalu dikurangkan
    4. \[\frac{4}{5} - \frac{10}{100} = \frac{4 \times 20}{5 \times 20} - \frac{10}{100}\]

    = \[\frac{80}{100} - \frac{10}{100} = \frac{70}{100}\]

  4. kita sederhanakan
    5. \[\frac{70:10}{100:10} = \frac{7}{10}\]

    maka sisa kue ibu adalah \[\frac{7}{10}\]
