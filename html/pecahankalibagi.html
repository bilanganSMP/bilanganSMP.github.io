Perkalian Bilangan pecahan

Misalkan ibu mempunyai kue amparan tatak \[\frac{1}{2}\] bagian. \[\frac{1}{3}\] bagiannya diberikan kepada kakak. jika bagian kue amparan tatak yang diberikan kepada kakak dibandingkan dengan sebuah kue amparan tatak yang utuh, berapa bagiankah yang diberikan kepada kakak? Perhatikan gambar di bawah ini.



Dari gambar di atas terlihat bahwa kue yang diberikan kepada kaka dibandingkan dengan kue keseluruhan adalah \[\frac{1}{6}\] bagian. Sehingga, \[\frac{1}{3}\] dari \[\frac{1}{2}\] adalah \[\frac{1}{6}\],

atau \[\frac{1}{3} \times \frac{1}{2} \]=\[\frac{1 \times 1}{3 \times 2}\]=\[\frac{1}{6}\]


Kesimpulan
Misalkan diketahui Pecahan \[\frac{a}{b}\] dan \[\frac{a}{b}\] dengan b ≠ 0 dan d ≠ 0, maka :

\[\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}\]

Dari kesimpulan tersebut diperoleh :

Untuk b = 1 diperoleh

\[\frac{a}{1}\times \frac{c}{d} = \frac{a\times c}{b\times d} \Leftrightarrow a\times \frac{c}{d} = \frac{a\times c}{d}\]

untuk lebih jelasnya cara mengalikan pecahan lihatlah animasi dbawah berikut ini

Petunjuk
  • tekan tombol mulai pada animasi dibawah
  • jika animasi belum muncul harap tunggu sebentar


Untuk Perkalian desimal dapat dilakukan dengan cara berikut:
Contoh:
  1. 2,83 x 1,6 =…
    2. \[ 2,83 \times 1,6 = \frac{283}{100} \times \frac{16}{10}\]

    \[= \frac{283 \times 16}{100 \times 10} = \frac{4528}{1000} = 4,528\]

  2. 32,3 x 0,4 =…
    3. \[32,3 \times 0,4 = \frac{323}{10} \times \frac{4}{10}\]

    \[= \frac{323 \times 4}{10 \times 10} = \frac{1292}{100} = 12,92\]

Pembagian Bilangan Pecahan

Pada operasi hitung Bilangan bulat, diketahui bahwa Pembagian merupakan operasi kebalikan dari Perkalian. Bagaimana operasi Pembagian pada Bilangan Pecahan?. Perhatikan Perkalian Bilangan bulat dengan Pecahan berikut.

  1. \[3 \times \frac{1}{2} = \frac{3 \times 1}{2} = \frac{3}{2} = 3 ∶ 2\] sehingga \[3 ∶ 2 = \frac{3}{2} = 3 \times \frac{1}{2}\]

  2. \[5 \times \frac{1}{3} = \frac{5 \times 1}{3} = \frac{5}{3} = 5 ∶ 3\] sehingga \[5 ∶ 3 = \frac{5}{3} = 5 \times \frac{1}{3}\]
    3. jadi dapat disimpulkan bahwa : \[a : b = a \times \frac{1}{b}\]

Kesimpulan
Misalkan \[a = \frac{m}{n}\] dan \[b = \frac{p}{q}\] maka diperoleh:

\[\frac{m}{n} ∶ \frac{p}{q} = \frac{m \times q}{n \times p}\]

untuk lebih jelasnya cara membagi pecahan lihatlah animasi dbawah berikut ini

Petunjuk
  • tekan tombol mulai pada animasi dibawah
  • jika animasi belum muncul harap tunggu sebentar


Pembagian decimal dapat dilakukan dengan cara berikut:
Contoh:
  1. 2,64 : 1,2 =…
    2. \[ 2,64 : 1,2 = \frac{264}{100} : \frac{12}{10}\]

    \[= \frac{264}{100} \times \frac{10}{12} = \frac{2640}{1200} = 2,2\]

    jadi 2,64 : 1,2 = 2,2

  2. 1,33 : 0,3 =…
    3. \[ 1,33 : 0,3 = \frac{133}{100} : \frac{3}{10} \]

    \[= \frac{133}{100} \times \frac{10}{3} = \frac{1330}{300} = 4,43\]

    jadi 1,33 : 0,3 = 4,43

1 dari 2
tentukan hasil dari \[3\frac{2}{5} \times \] 15%
penyelesaian:
  1. kita ubah semua pecahan diatas menjadi pecahan biasa

    2. \[\frac{3 \times 5 + 2}{5} \times \frac{15}{100}\]

    = \[\frac{17}{5} \times \frac{15}{100}\]

  2. kita kalikan

    3. \[\frac{17}{5} \times \frac{15}{100} = \frac{255}{500}\]

  3. selanjutnya kita sederhanakan

    4. \[\frac{255:5}{500:5} = \frac{51}{100}\]

2 dari 2
tentukan hasil dari \[6\frac{2}{5} : 0,6\]
penyelesaian:
  1. kita ubah semua pecahan diatas menjadi pecahan biasa

    2. \[\frac{6 \times 5 + 2}{5} : \frac{6}{10}\]

    = \[\frac{32}{5} : \frac{6}{10}\]

  2. kita balik pecahan kedua dan ubah operasi bagi menjadi kali

    3. \[\frac{32}{5} : \frac{6}{10} = \frac{32}{5} \times \frac{10}{6}\]

  3. kita kalikan dan sederhanakan

    4. \[\frac{32}{5} \times \frac{10}{6} = \frac{320}{30} = \frac{320:10}{30:10}\]

    \[=\frac{32}{3}\]

    atau
    bisa kita sederhanakan dulu sebelum dikali

    \[\frac{32:2}{5:5} \times \frac{10:5}{6:2} = \frac{16}{1} \times \frac{2}{3}\]

    \[=\frac{32}{3}\]

  4. kita sederhanakan lagi menjadi pecahan campuran
    5. \[=\frac{32}{3} = \frac{30}{3} + \frac{2}{3} = 10\frac{2}{3} \]

    jadi hasil dari \[6\frac{2}{5} : 0,6 = 10\frac{2}{3} \]
